# Gutenberg Books API - User Guide

## Overview
A professional REST API for accessing Project Gutenberg's digital book collection. This API provides comprehensive search, filtering, and pagination capabilities for exploring over 70,000 free eBooks.

## Live Deployment
- **Health Check**: https://gutenberg-books-api-b5qi.onrender.com

- **API Base URL**: https://gutenberg-books-api-b5qi.onrender.com/api/books

- **Source Code**: https://github.com/rishabh-kusram/gutenberg-books-api

## Quick Start
1. **Browser**: Navigate to any endpoint URL
2. **API Client**: Use Postman, Insomnia, or curl
3. **Integration**: Make HTTP GET requests to the endpoints
4. **Authentication**: None required - fully public API

## API Endpoints

### Base Endpoint
```
GET /api/books
```
Returns paginated list of all books (25 per page)

### Query Parameters
| Parameter | Type | Description | Example |
|-----------|------|-------------|---------|
| `page` | integer | Page number (default: 1) | `?page=2` |
| `language` | string | Language codes (comma-separated) | `?language=en,fr` |
| `title` | string | Search in book titles | `?title=pride` |
| `author` | string | Search in author names | `?author=dickens` |
| `topic` | string | Search in subjects/bookshelves | `?topic=fiction` |

## Usage Examples

### Basic Usage
```bash
# Get all books (first page)
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books"

# Get specific page
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?page=3"
```

### Filtering Examples
```bash
# Filter by language
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?language=en"

# Search by author
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?author=shakespeare"

# Search by title
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?title=alice"

# Filter by topic
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?topic=mystery"
```

### Advanced Filtering
```bash
# Multiple filters
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?language=en&topic=fiction&page=2"

# Multiple languages
curl "https://gutenberg-books-api-b5qi.onrender.com/api/books?language=en,fr"
```
