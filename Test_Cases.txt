# Gutenberg Books API - Test Cases

## Test Environment
- Base URL : https://gutenberg-books-api-b5qi.onrender.com/api/books
- Method : GET requests only
- Authentication : None required

---

## Test Cases

### 1. Fetch All Books
**Purpose**: Verify basic API functionality
**Endpoint**: '/api/books'
**CURL Command**: 

curl -X GET "https://gutenberg-books-api-b5qi.onrender.com/api/books"

**Expected Result**:
- HTTP Status: 200 OK
- Response contains 'count' and 'books' array
- Books array has up to 25 items
- Each book has id, title, download_count, authors, languages, subjects, bookshelves, formats

---
